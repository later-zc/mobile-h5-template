# 概述 OverView

基于 `jQuery` + `原生html/css/js` 构建的移动端 `h5` 项目模板

- 项目入口：`index.html`

- 入口样式：`index.css`

- 公共样式：`common.css`

- 重置样式：`reset.css`

- 页面样式：`page/*.css`

- 业务逻辑：`main.js`

- 工具函数：`util.js`

- 公共依赖 `js`：`common/*.js`

- 开发依赖 `js`：`dev/.js`

- 资源文件：`assets/*`

# 特性 Speciality
 
- 使用 `vscode` 扩展插件 `easy less` 在保存 `.less` 文件时，将 `.less` 文件编译压缩成同名 `.css` 文件

- 使用 `jQuery` 插件 `jquery-mockjax` 来模拟请求响应数据 https://github.com/jakerella/jquery-mockjax
